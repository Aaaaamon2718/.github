## Month 4: データ整理・ラベリング

### 4.1 ラベリングスキーマ

| カラム | データ定義 | 設定値例 |
|--------|----------|---------|
| ID | 一意識別番号 | `QA_001`, `VID_202405_01_05` |
| カテゴリ | 大分類 | 法人保険, ドクター, 相続, 営業マインド |
| サブカテゴリ | 詳細分類 | 決算書分析, 退職金, アプローチ, クロージング |
| 質問/トピック | 質問文またはトピック主題 | 「赤字決算の社長へのアプローチ方法は？」 |
| 回答/内容 | 牧野氏の回答テキスト | （文字起こしデータに基づく） |
| 出典 | 情報ソース元 | 牧野生保塾 Vol.5 (2024/05) |
| 言い回しタグ | 特徴的表現 | 断定, 比喩表現, 関西弁, 厳しい指摘 |
| 感情タグ | 発言時のトーン | 励まし, 叱咤, 論理的解説, 雑談 |
| 優先度 | 学習における重要度 | 高（頻出質問）, 中, 低 |

### 4.2 人格トレースデータ整理

| フレーズ・特徴 | 使用場面 | 頻出度 |
|--------------|---------|--------|
| 決めの言葉・座右の銘 | 研修の締め、励まし | 高 |
| 励まし・共感 | 成果が出ず悩んでいる時 | 高 |
| 叱咤・プロ意識 | 甘えが見える時 | 中 |
| 文末表現・口調 | 全般的な会話スタイル | 高 |

> **ポイント**: 単なる口癖の模倣だけでなく、「思考プロセス」と「価値観」をプロンプトに組み込み、
> 未知の質問に対しても「牧野氏ならどう考えるか」を推論させる。
