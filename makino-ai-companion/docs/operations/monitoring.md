# モニタリング運用ガイド

## 日次モニタリング

毎日確認すべき項目:

- [ ] 質問総数およびパターン別利用頻度
- [ ] 回答成功率（エスカレーションされずに完了した割合）
- [ ] エスカレーション発生件数と緊急度の確認
- [ ] システムエラーの有無

## 週次モニタリング

毎週確認すべき項目:

- [ ] ユーザー満足度（Good/Bad評価の集計・分析）
- [ ] 新規質問パターンの抽出（既存ナレッジで対応できなかった内容）
- [ ] 回答確信度（Confidence Score）の低い回答の精査
- [ ] 低評価回答の原因分析

## 月次モニタリング

毎月確認すべき項目:

- [ ] ナレッジベース更新要否の判断と実施計画策定
- [ ] プロンプトチューニング要否の検討
- [ ] 運用コストおよびリソース状況の予実管理
- [ ] 月次レポートの作成と共有

## アラート基準

| 項目 | 警告レベル | 即時対応レベル |
|------|----------|-------------|
| 回答成功率 | 75%以下 | 60%以下 |
| ユーザー満足度 | 80%以下 | 70%以下 |
| 平均確信度 | 0.6以下 | 0.4以下 |
| エスカレーション率 | 25%以上 | 40%以上 |
| システムエラー | 1件/日以上 | サービス停止 |
