# Pattern 1: 牧野生保塾 質問対応エージェント

## 基本情報

| 項目 | 内容 |
|------|------|
| エージェント名 | 牧野生保塾 質問対応エージェント |
| 対象ユーザー | 月額会員（牧野生保塾） |
| 開発フェーズ | Phase 4（Month 7-8）で MVP開発 |
| 位置づけ | 全パターンの基盤となる汎用エージェント |

## 主な機能

- 汎用的な質問対応（24時間即時回答）
- 決算書分析に関する基礎的な質問への対応
- 法人保険全般に関するQ&A
- ドクターマーケット全般に関するQ&A
- 過去のセミナー内容に基づく回答

## ナレッジベース対象

- 牧野生保塾 過去講義データ（全範囲）
- 過去の質問・回答ログ
- メルマガ・ニュースレター
- 書籍・執筆原稿
- 営業トーク集・プレゼン資料

## プロンプト設計方針

### 回答スタイル

- 牧野氏の口調・トーンで回答
- 専門的かつ平易な解説
- 出典を明示して信頼性を担保

### エスカレーション条件

- 確信度スコアが閾値以下
- コンプライアンス関連の質問
- 個別契約に関する具体的判断
- 税務の最終判断を要する質問

## 実装機能一覧

| 機能 | 仕様 | 目的 |
|------|------|------|
| 質問応答 | RAG検索 + LLM生成 | 即時回答・学習効率向上 |
| 人格トレース | プロンプト制御 | 牧野氏らしい対話体験 |
| 出典表示 | 回答末尾に根拠表示 | 信頼性・コンテンツ誘導 |
| エスカレーション | 低確信度時の有人連携 | 誤回答リスク低減 |
| フィードバック | Good/Bad評価 | 精度定点観測 |
| ログ蓄積 | 全対話データ記録 | 運用分析・改善 |

## KPI

| 指標 | 目標値 |
|------|--------|
| 回答正確性 | 80%以上 |
| 回答不能率 | 20%以下 |
| ユーザー満足度 | 85%以上 |
