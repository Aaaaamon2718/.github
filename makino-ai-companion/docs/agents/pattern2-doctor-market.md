# Pattern 2: ドクターマーケット特化エージェント

## 基本情報

| 項目 | 内容 |
|------|------|
| エージェント名 | ドクターマーケット特化エージェント |
| 対象ユーザー | 開業医攻略研修 受講生 |
| 開発フェーズ | Phase 5（Month 9）で開発 |
| 位置づけ | 医療領域に特化した専門エージェント |

## 主な機能

- 医療法人・個人開業医へのアプローチ手法
- 医療法、特有の税制に関する知識提供
- ドクターとの面談ロールプレイング
- 多忙なドクターの関心を引く営業トーク

## ナレッジベース対象

- 開業医攻略研修データ（動画・テキスト・Q&A）
- 医療法人固有の知識体系
- ドクター向け特化型営業トークスクリプト
- 医療業界の専門用語辞書

## プロンプト設計方針

### 文脈判別ロジック

質問者が対峙している相手が「医療法人」か「個人開業医」かを文脈から推論し、回答を最適化する。

### 専門用語の解釈精度

医療業界特有の用語や略語を正しく理解し、誤解のない回答を生成する。

### 信頼獲得プロセス

ドクターに「知識がある」と感じさせる、論理的かつ専門的な言い回しを適用する。

## 境界条件（Pattern 1との棲み分け）

| 領域 | Pattern 2で対応 | Pattern 1にルーティング |
|------|----------------|----------------------|
| 医療法人の事業承継 | o | - |
| 開業医へのアプローチ | o | - |
| 一般的な法人保険 | - | o |
| 営業全般のマインド | - | o |
