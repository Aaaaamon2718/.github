# Pattern 3: 法人保険特化エージェント

## 基本情報

| 項目 | 内容 |
|------|------|
| エージェント名 | 法人保険特化エージェント |
| 対象ユーザー | 法人財務スペシャリスト研修 受講生 |
| 開発フェーズ | Phase 5（Month 9）で開発 |
| 位置づけ | 財務・税務に特化した専門エージェント |

## 主な機能

- 高度な決算書分析と財務改善提案
- 退職金設計、事業承継スキーム
- 税制改正に対応した最新情報の提供
- 数値に基づく論理的な提案フロー

## ナレッジベース対象

- 法人財務スペシャリスト研修データ（録画・テキスト）
- 決算書分析知識（P/L、B/S、財務指標）
- 退職金準備・事業承継対策の事例データ
- 税制関連の最新情報

## プロンプト設計方針

### 数値処理の厳格化

- ハルシネーション（嘘の数値生成）を抑制
- 計算根拠を明確に示す
- 不確実な数値には必ず注釈を付与

### 税制対応の正確性

- 税務に関する回答には必ず「税理士への確認」を促す注記を付与
- 税率・控除額等の具体的数値は最新データに基づくことを明示
- 確定情報と見解の区別を明確にする

### 論理的構成

回答は以下のフローに沿って生成:

```
1. 現状分析（決算書データの読み解き）
2. 課題特定（財務上の問題点の指摘）
3. 解決策提示（具体的な改善提案）
```

## 境界条件

| 領域 | Pattern 3で対応 | Pattern 1にルーティング |
|------|----------------|----------------------|
| 決算書の詳細分析 | o | - |
| 退職金設計スキーム | o | - |
| 事業承継の専門相談 | o | - |
| 一般的な営業手法 | - | o |
| メンタルサポート | - | Pattern 4 |

## 重要な制約

- **具体的な保険料試算は行わない**（エスカレーション対象）
- **税務の最終判断は必ず「税理士に確認」を促す**
- **個別の契約内容に関する助言は行わない**

## KPI

| 指標 | 目標値 |
|------|--------|
| 数値の正確性 | 95%以上 |
| 回答正確性 | 80%以上 |
| 税務注記付与率 | 100% |
| ユーザー満足度 | 85%以上 |
