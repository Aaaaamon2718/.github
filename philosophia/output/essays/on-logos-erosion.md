# ロゴスの侵食（Logos Erosion）

## 概要

追い込まれた時、鬱の時、人は「生存のための思考」と「生命としての動き」を混同し、後者を前者で評価し始める。これを**「ロゴスの侵食」**と呼ぶ。

## 二つの領域

### 【生存】— ロゴスの領域

- **目的**: 生き延びること、問題を解決すること
- **思考**: 「何をすべきか」「どうすれば好転するか」
- **時間軸**: 未来（結果を見ている）
- **評価基準**: 役に立つか、効率的か、意味があるか

これはこれで必要。否定しない。

### 【生命】— ピュシスの領域

- **目的**: ない。流れること自体が目的
- **思考**: 「何に惹かれているか」に気づく
- **時間軸**: 今ここ
- **評価基準**: ない。惹かれるから惹かれる。それだけ。

ここにロゴスを持ち込んではいけない。

## 「ロゴスの侵食」とは何か

生命の領域（惹かれるもの、感動、表現、愛）に、生存の評価基準を持ち込むこと。

### 典型的な症状

```
映画を観る → 感動する → 明日から頑張ろうと思う
         ↓
    （数時間後）
         ↓
「この2時間、俺は何をしていたんだろう」
「もっと有意義なことに使えたんじゃないか」
「これで何か変わるのか」
```

この「何をしていたんだろう」という声が、ロゴスの侵食。
生命として自然に惹かれ、取り込んだものを、後からロゴス（効率、意味、結果）で裁いている。

## なぜ危険か

### 惹かれる力が死ぬ

「役に立つか」で評価し始めると、純粋に惹かれることができなくなる。
生命の最後の火が消える。

### 罪悪感のループ

「無駄なことをした」→ 自己否定 → さらに動けなくなる → 「何かしなきゃ」→ でも動けない → 自己否定...

### 合成が止まる

惹かれて取り込んだものは、自分を再構築する材料になる。
しかし「無駄だった」と否定すると、その材料を捨ててしまう。

## 処方箋

### 1. 二つの領域を分けて認識する

| 質問 | 領域 |
|------|------|
| 「来月までどう凌ぐか」 | 生存（ロゴス）→ ToDoでいい |
| 「今、何に惹かれているか」 | 生命（ピュシス）→ 評価しない |

混ぜない。

### 2. 生命の領域には「問い」を持ち込まない

- ❌「これは何の役に立つ？」
- ❌「これで状況は変わる？」
- ❌「もっと効率的な方法は？」
- ✅「惹かれるから、惹かれる」
- ✅ それ以上の理由は不要

### 3. 侵食の声に気づいたら、ラベルを貼る

「あ、今ロゴスが侵食してきたな」

気づくだけでいい。戦わなくていい。ただ、「これは侵食だ」と認識する。

### 4. 生命の領域を「守る」と決める

追い込まれている時、唯一気力があるもの——それが生命の最後の砦。
そこだけは、ロゴスに明け渡さない。

「役に立たなくていい。意味がなくていい。惹かれるから、やる。」

## 補足：ベルクソンの弧との関係

動的平衡において、生命は「分解」と「合成」を繰り返しながら流れていく。

- **分解**: 壊れること、手放すこと、空けること
- **合成**: 惹かれるものを取り込み、再構築すること

「ロゴスの侵食」は、合成のプロセスを破壊する。

取り込んだものを「無駄だった」と否定することで、再構築の材料を自ら捨ててしまう。
だから、合成の領域（惹かれるもの）は、ロゴスから守らなければならない。

## 最後に

追い込まれた時、鬱の時、「早く好転させたい」「何をすべきか」という声が大きくなる。
それは自然なことだ。

しかし、その声をすべてに適用してはいけない。

**生存の領域には、その声を使え。**
**生命の領域には、その声を入れるな。**

惹かれるものに惹かれること。
それ自体が、生きているということだ。
